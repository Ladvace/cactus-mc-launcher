# Screenshots

Generated with the capture script — no need to place files by hand.

```bash
bun run tauri dev            # app running & visible
bun run shot home            # navigate to the view, then run this
bun run shot themes          # (home, themes, servers, run-a-server,
bun run shot servers         #  community, accounts, groups, settings)
```

Each run captures the current Cactus Launcher window into `docs/screenshots/<name>.png`,
which the root `README.md` references. macOS only; needs Screen Recording permission
for your terminal (System Settings → Privacy & Security → Screen Recording).
